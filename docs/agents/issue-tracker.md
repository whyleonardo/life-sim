# Issue Tracker

Issues are tracked on **GitHub** in the [whyleonardo/life-sim](https://github.com/whyleonardo/life-sim) repository.

## Workflow

- **Create issues**: `gh issue create`
- **List issues**: `gh issue list`
- **View issue**: `gh issue view <number>`
- **Close issue**: `gh issue close <number>`
- **Add labels**: `gh issue edit <number> --add-label <label>`

## Labels

See [triage-labels.md](./triage-labels.md) for the canonical label vocabulary.
